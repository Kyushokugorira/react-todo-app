# TodoApp

React、TypeScript、Tailwind CSS を使用し、ローカルストレージでデータを永続化した「Todoアプリ」です。

## 開発履歴

- 2024年10月24日：プロジェクト開始

## 機能

- 名前入力
- Todoの追加
- - 名前、優先度、期限の入力
- 完了したTodoの削除
- 未完了タスクの表示
- Todoのソート

## 工夫ポイント

- 名前の入力ができるようにした
- 優先度が高い順、期限が早い順に並べ替えられるようにした
